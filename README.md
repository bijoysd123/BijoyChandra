# REEL — মোশন গ্রাফিক্স ও ভিডিও এডিটিং পোর্টফোলিও

## ফাইলগুলো
- `index.html` — মূল পেজ
- `style.css` — ডিজাইন
- `script.js` — প্রজেক্ট তালিকা, ভিডিও মোডাল, স্ক্রুবার অ্যানিমেশন

## ১. ফ্রিতে হোস্ট করবেন কীভাবে (কোনো ডোমেইন/পেমেন্ট লাগবে না)

### Netlify Drop (সবচেয়ে সহজ, ২ মিনিট)
1. https://app.netlify.com/drop -এ যান
2. `portfolio` ফোল্ডারটা (এই তিনটা ফাইল সহ) পুরোটা ড্র্যাগ করে ওখানে ছেড়ে দিন
3. সাথে সাথে একটা ফ্রি লিংক পাবেন, যেমন `yourname.netlify.app`
4. চাইলে সাইট Settings থেকে সাবডোমেইনের নাম বদলাতে পারবেন (এখনও ফ্রি)

### GitHub Pages (বিকল্প)
1. GitHub-এ একটা নতুন রিপোজিটরি বানান
2. এই ফাইলগুলো আপলোড করুন
3. Settings → Pages → branch সিলেক্ট করে Save করুন
4. `username.github.io/repo-name` লিংকে সাইট লাইভ হবে

## ২. নিজের ভিডিও/প্রজেক্ট যোগ করবেন কীভাবে

`script.js` ফাইলের একদম উপরে `PROJECTS` নামের লিস্ট আছে। প্রতিটা প্রজেক্টের জন্য:

```js
{ title: "আপনার প্রজেক্টের নাম", category: "Motion Graphics", youtubeId: "dQw4w9WgXcQ", colors: ["#35D0C0", "#FF6B35"] }
```

- **youtubeId** বের করার নিয়ম: আপনার ইউটিউব লিংক যদি হয়
  `https://www.youtube.com/watch?v=dQw4w9WgXcQ`
  তাহলে `=` চিহ্নের পরের অংশটাই (`dQw4w9WgXcQ`) আপনার `youtubeId`।
- `youtubeId` খালি (`""`) রাখলে সেই প্রজেক্টে শুধু প্লেসহোল্ডার থাম্বনেইল দেখাবে, ক্লিকে কিছু হবে না — যতক্ষণ না আপনি আসল ID বসান।
- নতুন প্রজেক্ট যোগ করতে চাইলে লিস্টে নতুন লাইন `{ ... }` যোগ করুন, কমা দিয়ে আলাদা করে।

## ৩. অন্যান্য কাস্টমাইজেশন

- **ইমেইল/সোশ্যাল লিংক**: `index.html`-এর `#contact` সেকশনে গিয়ে `your@email.com` ও সোশ্যাল লিংকগুলো বদলে দিন।
- **টুলস/সফটওয়্যার লিস্ট**: `script.js`-এর `TOOLS` লিস্ট থেকে এডিট করুন।
- **কালার থিম**: `style.css`-এর একদম উপরে `:root` অংশে `--teal` আর `--orange` ভ্যারিয়েবল বদলালে পুরো সাইটের রং বদলে যাবে।
- **রিয়েল থাম্বনেইল ছবি**: আপাতত গ্রেডিয়েন্ট প্লেসহোল্ডার আছে। আসল ছবি বসাতে চাইলে বলবেন, আমি `<img>` দিয়ে বদলে দেব।

কোনো অংশ বদলাতে সমস্যা হলে, শুধু বলুন কী বদলাতে চান — আমি সরাসরি ফাইলেই করে দেব।
