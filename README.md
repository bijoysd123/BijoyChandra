# REEL — মোশন গ্রাফিক্স ও ভিডিও এডিটিং পোর্টফোলিও

## ফাইলগুলো
- `index.html` — মূল পেজ
- `style.css` — ডিজাইন
- `script.js` — প্রজেক্ট তালিকা, ভিডিও মোডাল, স্ক্রুবার অ্যানিমেশন

## ১. ফ্রিতে হোস্ট করবেন কীভাবে (কোনো ডোমেইন/পেমেন্ট লাগবে না)

### Netlify Drop (সবচেয়ে সহজ, ২ মিনিট)
1. https://app.netlify.com/drop -এ যান
2. `portfolio` ফোল্ডারটা (এই তিনটা ফাইল সহ) পুরোটা ড্র্যাগ করে ওখানে ছেড়ে দিন
3. সাথে সাথে একটা ফ্রি লিংক পাবেন, যেমন `yourname.netlify.app`
4. চাইলে সাইট Settings থেকে সাবডোমেইনের নাম বদলাতে পারবেন (এখনও ফ্রি)

### GitHub Pages (বিকল্প)
1. GitHub-এ একটা নতুন রিপোজিটরি বানান
2. এই ফাইলগুলো আপলোড করুন
3. Settings → Pages → branch সিলেক্ট করে Save করুন
4. `username.github.io/repo-name` লিংকে সাইট লাইভ হবে

## ২. নিজের ভিডিও/প্রজেক্ট যোগ করবেন কীভাবে

`script.js` ফাইলের একদম উপরে `PROJECTS` নামের লিস্ট আছে। প্রতিটা প্রজেক্টের জন্য:

```js
{ title: "আপনার প্রজেক্টের নাম", category: "Motion Graphics", youtubeId: "dQw4w9WgXcQ", colors: ["#35D0C0", "#FF6B35"] }
```

- **youtubeId** বের করার নিয়ম: আপনার ইউটিউব লিংক যদি হয়
  `https://www.youtube.com/watch?v=dQw4w9WgXcQ`
  তাহলে `=` চিহ্নের পরের অংশটাই (`dQw4w9WgXcQ`) আপনার `youtubeId`।
- `youtubeId` খালি (`""`) রাখলে সেই প্রজেক্টে শুধু প্লেসহোল্ডার থাম্বনেইল দেখাবে, ক্লিকে কিছু হবে না — যতক্ষণ না আপনি আসল ID বসান।
- নতুন প্রজেক্ট যোগ করতে চাইলে লিস্টে নতুন লাইন `{ ... }` যোগ করুন, কমা দিয়ে আলাদা করে।

## ৩. থাম্বনেইলে ৫-৬ সেকেন্ডের অটো-প্লে ভিডিও প্রিভিউ যোগ করা

কার্ডের থাম্বনেইলে এখন একটা ছোট ভিডিও লুপ হয়ে চলতে পারে (YouTube না — নিজের আপলোড করা ছোট, লো-কোয়ালিটি mp4 ফাইল, যাতে দ্রুত লোড হয়)।

### ধাপ ১ — ভিডিও ক্লিপ বানান (৫-৬ সেকেন্ড, ছোট সাইজ)
আপনার আসল ভিডিও থেকে ৫-৬ সেকেন্ডের একটা অংশ কেটে, রেজোলিউশন কমিয়ে, সাউন্ড বাদ দিয়ে একটা হালকা mp4 বানাতে হবে। ফ্রি টুল দিয়ে করতে পারেন:

- **অনলাইন (সহজ)**: [Clideo](https://clideo.com/compress-video) বা [FreeConvert](https://www.freeconvert.com/video-compressor) দিয়ে ভিডিও ছোট করে ডাউনলোড করুন
- **HandBrake (ফ্রি সফটওয়্যার)**: https://handbrake.fr — ট্রিম + কম্প্রেস দুটোই এক জায়গায় করা যায়
- **কমান্ড লাইনে (ffmpeg থাকলে)**:
  ```
  ffmpeg -i input.mp4 -t 6 -vf "scale=480:-1" -an -c:v libx264 -crf 32 -preset veryslow -movflags +faststart output.mp4
  ```
  এটা ৬ সেকেন্ড কেটে, ৪৮০px চওড়ায় ছোট করে, সাউন্ড বাদ দিয়ে, হালকা ফাইল বানাবে (সাধারণত ৫০০KB–২MB এর মধ্যে)।

**লক্ষ্য রাখবেন:** প্রতিটা প্রিভিউ ফাইল যত সম্ভব ছোট (আদর্শভাবে ৩ MB-এর নিচে) রাখলে সাইট দ্রুত লোড হবে।

### ধাপ ২ — ফাইল আপলোড করুন
GitHub-এ `assets/previews/` ফোল্ডারে গিয়ে **Add file → Upload files** দিয়ে আপনার mp4 ফাইলগুলো আপলোড করুন (যেমন `aurora.mp4`)।

### ধাপ ৩ — script.js-এ পাথ বসান
`PROJECTS` লিস্টে সংশ্লিষ্ট প্রজেক্টের `previewVideo` ফিল্ডে পাথ বসান:

```js
{ title: "Brand Reel — Aurora", category: "Motion Graphics", youtubeId: "fRoOwRC-JiE", previewVideo: "assets/previews/aurora.mp4", colors: ["#35D0C0", "#0A0B0D"] },
```

খালি (`""`) রাখলে শুধু গ্রেডিয়েন্ট থাম্বনেইল দেখাবে, ভিডিও চলবে না।

**কীভাবে কাজ করে:** থাম্বনেইল স্ক্রিনে দেখা গেলেই ভিডিও নিজে থেকে (সাউন্ড ছাড়া) লুপ হয়ে চলবে; স্ক্রল করে সরে গেলে অটোমেটিক পজ হয়ে যাবে (এতে সাইট স্লো হবে না)। ক্লিক করলে আগের মতোই YouTube ভিডিও মোডালে চালু হবে।

## ৪. ভিডিও মোডাল (পপ-আপ) কাস্টমাইজেশন

কার্ডে ক্লিক করলে যে ভিডিও বক্সটা খোলে, তার লুক এখন সম্পূর্ণ কাস্টমাইজেবল। `style.css`-এর একদম উপরে `:root` অংশে এই ৫টা ভ্যারিয়েবল আছে:

```css
--modal-overlay: radial-gradient(...), radial-gradient(...), rgba(10,11,13,0.86); /* পেছনের আস্তরণের রং */
--modal-blur: 14px;        /* পেছনের কনটেন্ট কতটা ঝাপসা হবে */
--modal-radius: 20px;      /* ভিডিও বক্সের কোণা কতটা রাউন্ড হবে */
--modal-max-width: 960px;  /* ভিডিও বক্সের সর্বোচ্চ প্রস্থ (বড় স্ক্রিনে) */
--modal-width-vw: 88vw;    /* ভিডিও বক্স স্ক্রিনের কত শতাংশ জায়গা নেবে */
```

- **কোণা রাউন্ড করা**: `--modal-radius`-এর মান বাড়ালে/কমালে (যেমন `0px` = একদম চারকোণা, `32px` = খুব গোল)।
- **ব্যাকগ্রাউন্ডের রং**: `--modal-overlay`-তে চাইলে একদম সাদামাটা কালো রাখতে পারেন — `rgba(10,11,13,0.94)` — অথবা টিল-অরেঞ্জ গ্রেডিয়েন্ট রাখতে পারেন (এখনকার ডিফল্ট), যেকোনো hex/rgba রং বসাতে পারেন।
- **ব্যাকগ্রাউন্ড ব্লার**: `--modal-blur: 0px` দিলে ব্লার বন্ধ হয়ে যাবে; সংখ্যা যত বাড়াবেন, পেছনের কার্ড তত ঝাপসা দেখাবে।
- **ভিডিওর সাইজ**: `--modal-width-vw` দিয়ে স্ক্রিনের কত শতাংশ জায়গা নেবে ঠিক করুন (যেমন `95vw` = প্রায় ফুল স্ক্রিন), আর `--modal-max-width` দিয়ে বড় মনিটরে এটা কতটার বেশি চওড়া হবে না তা বেঁধে দিন।

## ৫. অন্যান্য কাস্টমাইজেশন

- **ইমেইল/সোশ্যাল লিংক**: `index.html`-এর `#contact` সেকশনে গিয়ে `your@email.com` ও সোশ্যাল লিংকগুলো বদলে দিন।
- **টুলস/সফটওয়্যার লিস্ট**: `script.js`-এর `TOOLS` লিস্ট থেকে এডিট করুন।
- **কালার থিম**: `style.css`-এর একদম উপরে `:root` অংশে `--teal` আর `--orange` ভ্যারিয়েবল বদলালে পুরো সাইটের রং বদলে যাবে।
- **রিয়েল থাম্বনেইল ছবি**: আপাতত গ্রেডিয়েন্ট প্লেসহোল্ডার আছে। আসল ছবি বসাতে চাইলে বলবেন, আমি `<img>` দিয়ে বদলে দেব।

কোনো অংশ বদলাতে সমস্যা হলে, শুধু বলুন কী বদলাতে চান — আমি সরাসরি ফাইলেই করে দেব।
